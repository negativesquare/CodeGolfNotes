# Code Golf Notes for AtCoder (by negativesquare)

AtCoder でコードゴルフをしているうちに溜まった個人メモをまとめたものです。
せっかくなので公開していますが、**コードゴルフ前提の書き方になっているため可読性は低い**です。

## 注意点

- **各ファイルには複数のコード片が含まれています。**
  そのため、ファイル全体をそのまま実行するとほぼ間違いなくエラーになります。
  **必要な部分だけをコピーし、問題に応じて整形して使う想定です。**

- **元々は自分用メモのため、未検証の記述や古い記述が残っています。**
  すでにより短い書き方を見つけていても、修正が追いついていない箇所があります。

## 構成

言語ごとにフォルダを分けています(現時点では Python のみ)。
各フォルダ内の README.md に、ファイルごとの補足説明を記載しています。

## 付属ファイル

この README.md と同階層に **`exitcode.csv`** を置いています。
これは **AtCoder の提出/コードテスト結果(終了コード・ステータス)** を一覧表にまとめたものです。
